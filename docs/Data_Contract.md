Таблица праздников в Германии.
**Зерно таблицы:** Одна строка = один праздничный день в Германии.
**Путь к файлу:** `data/normalized/variant_20/YYYY-MM-DD_HH-MM-SS_holidays_germany.csv`

### Схема данных (Schema)

| Поле | Тип данных | Nullable | Описание |
| :--- | :--- | :--- | :--- |
| **holiday_date** | datetime | No | Точная дата праздника в формате ГГГГ-ММ-ДД |
| **local_name** | string | No | Официальное название праздника на немецком языке |
| **eng_name** | string | No | Название праздника на английском языке |
| **category** | string | No | Категория (Public, Optional и т.д.) |

### Логика очистки (Cleaning Logic)
1. **Типизация:** Поле `date` приведено из строкового типа к формату `datetime64[ns]`.
2. **Раскрытие списков:** Из поля `types` извлечено первое значение (строка) вместо списка.
3. **Фильтрация:** Удалены технические поля API (`fixed`, `global`, `counties`, `launchYear`), не несущие ценности для базового анализа.
