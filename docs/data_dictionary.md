# Словарь данных (Data Dictionary)

Этот документ описывает смысл полей в сквозном проекте по анализу праздников Германии.

## 1. Витрина: monthly_data (Mart)
**Описание:** Агрегированные данные по количеству праздников в месяц. Сформирована с помощью группировки по номеру месяца `df.groupby('month_num').size()`.

| Поле | Бизнес-смысл | Пример | Примечание |
| :--- | :--- | :--- | :--- |
| **month_num** | Порядковый номер месяца (1-12). | `1` (Январь) | Извлечено из даты через `.dt.month`. |
| **count** (или значения size) | Общее количество праздничных дней в указанном месяце. | `2` | Результат функции агрегации `.size()`. |

## 2. Слой: Таблица праздников (Normalized)
**Описание:** Очищенный список праздников. Зерно: одна строка = один праздник.

| Поле | Бизнес-смысл | Пример |
| :--- | :--- | :--- |
| **date** | Календарная дата праздника в формате ГГГГ-ММ-ДД. | `2025-01-01` |
| **local_name** | Официальное название праздника на немецком языке. | `Neujahr` |
| **eng_name** | Название праздника на английском языке. | `New Year's Day` |
| **category** | Тип праздника (например, Public — государственный). | `Public` |
